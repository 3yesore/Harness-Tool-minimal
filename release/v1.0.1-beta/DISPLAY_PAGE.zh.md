# Harness Tool Minimal v1.0.1 历史基线展示页

## 版本定位

`Harness Tool Minimal v1.0.1 beta` 对应的是一条已经冻结的 minimal 基线。  
它不是厚平台，也不是功能堆叠版，而是一个适合长期作为开发框架底座使用的历史版本展示页。

## 这版展示什么

- 最小闭环：`init / apply / validate`
- 标准化模块入口：`INDEX.md`、`SPEC.md`、`CHANGELOG.md`
- 轻量扩展面：`profiles/`、`templates/`、`.openclaw_skill/`
- 默认一致性模式：`soft`
- 扩展方式：示例优先、项目本地实现

## 为什么它重要

- 它把模块开发、验证、交接和记录收束到少量固定文件中。
- 它保留了足够大的项目级自定义空间。
- 它适合小项目、小模块、开发初期的持续使用。
- 它是后续扩展、实验和回流的历史基线。
