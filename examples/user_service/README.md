# User Service Example

这是一个中等复杂度的示例模块，展示 Harness 规范在真实场景中的应用。

## 模块说明

`user_service` 是一个用户管理服务模块，包含：
- 用户注册和认证
- 配置管理
- 错误处理
- 完整的测试覆盖

## 与 hello_world 的区别

| 特性 | hello_world | user_service |
|------|-------------|--------------|
| 复杂度 | 最小示例 | 中等复杂度 |
| 配置 | 单一配置文件 | 多环境配置 |
| 依赖 | 无 | 有外部依赖 |
| 错误处理 | 简单 | 完整的错误码体系 |
| 测试 | 基础冒烟测试 | 多场景测试 |

## 学习要点

1. **多文件组织**：如何在 INDEX.md 中索引多个关键文件
2. **依赖管理**：如何在 INDEX.md 中声明外部依赖
3. **配置分层**：如何管理开发/生产环境配置
4. **错误处理**：如何在 SPEC.md 中定义完整的错误码
5. **测试覆盖**：如何编写覆盖多场景的冒烟测试
